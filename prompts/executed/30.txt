# Leira v1.19 — Verification Prompt Drafter

## Mission

Implement a deterministic Verification Prompt Drafter.

Given a slice label, subject module, and expected guardrails, it renders a reviewable verification prompt.

It does not run tests.

It does not inspect files.

It does not verify anything.

It only drafts the prompt that a human or external worker can use to request verification.
