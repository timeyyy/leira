# Leira v3.1 — Human Decision Draft Projection

## Mission

Implement the deterministic **Human Decision Draft Projection**.

This is **not** a planner.

This is **not** a dispatcher.

This is **not** autonomous.

This is **not** a decision engine.

It is a deterministic read model that converts an Engineering Kernel Result into a reviewable Human Decision draft.

It never creates a Human Decision Record.

It never approves anything.

It merely prepares a deterministic draft for human review.

---

## Files

Create:

```text
leira/human_decision_draft/
    __init__.py
    draft.py
    test_human_decision_draft.py
```

Modify nothing else unless strictly necessary.
