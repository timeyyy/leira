# Leira v3.3 — Dispatch Draft Projection

## Mission

Implement the deterministic **Dispatch Draft Projection**.

This is **not** a dispatcher.

This is **not** execution.

This is **not** autonomous.

This is **not** a Dispatch Record.

It is a deterministic read model that converts an explicitly committed HumanDecision into a reviewable Dispatch Draft.

It prepares the next lifecycle step.

It never creates a DispatchRecord.

---

## Files

Create:

```text
leira/dispatch_draft/
    __init__.py
    draft.py
    test_dispatch_draft.py
```

Modify nothing else unless strictly necessary.
