# Leira v3.4 — Dispatch Commit

## Mission

Implement the deterministic **Dispatch Commit**.

This is **not** a dispatcher.

This is **not** execution.

This is **not** autonomous.

Its sole responsibility is to convert an explicitly approved DispatchDraft into a DispatchRecord.

The commit is explicitly invoked by a human.

Without an explicit commit request, nothing happens.

---

## Files

Create:

```text
leira/dispatch_commit/
    __init__.py
    commit.py
    test_dispatch_commit.py
```

Modify nothing else unless strictly necessary.

---

Create immutable dataclass:

```text
DispatchCommit
```

Fields:

```text

dispatch_record
```

No generated identifiers.

No timestamps.

No clocks.

No UUIDs.
