# Leira v4.0 — Dispatcher Kernel

## Mission

Implement the first deterministic **Dispatcher Kernel**.

This is **not** an execution engine.

This is **not** a planner.

This is **not** autonomous.

Its sole responsibility is to validate a committed DispatchRecord and return a deterministic DispatchPlan describing exactly what would be executed.

It never performs execution.

It never invokes tools.

It never spawns processes.

It never mutates project state.

It is the final deterministic boundary before the outside world.
